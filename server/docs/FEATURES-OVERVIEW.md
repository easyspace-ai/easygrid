# 功能总览

## 🎯 核心功能

### 1. 多维表格系统

LuckDB 提供了完整的多维表格数据库系统，支持：

- **空间（Space）**: 顶层组织单元，支持多租户
- **Base**: 空间内的数据容器，使用独立的Schema
- **表格（Table）**: Base内的数据表，每个表对应一个物理表
- **记录（Record）**: 表中的数据行，支持动态字段

### 2. 动态Schema

- **运行时创建表**: 无需预定义表结构
- **动态添加字段**: 支持运行时添加、修改、删除字段
- **字段类型转换**: 支持字段类型之间的转换

### 3. 视图系统

- **网格视图（Grid）**: 表格形式展示
- **看板视图（Kanban）**: 看板形式展示
- **表单视图（Form）**: 表单形式展示
- **过滤和排序**: 支持复杂的过滤和排序条件
- **分组**: 支持按字段分组

### 4. 实时协作

- **多用户同时编辑**: 支持多个用户同时编辑同一表格
- **实时同步**: 操作实时同步到所有客户端
- **冲突检测**: 自动检测和解决编辑冲突
- **版本控制**: 使用乐观锁机制

### 5. 权限系统

- **角色权限**: 支持Owner、Editor、Viewer等角色
- **细粒度控制**: 支持空间、Base、表、字段级别的权限
- **协作者管理**: 支持添加、删除、修改协作者

### 6. 附件管理

- **文件上传**: 支持多种文件类型上传
- **文件存储**: 支持本地存储和云存储（S3、OSS）
- **文件访问**: 支持安全的文件访问控制

## 🔧 技术特性

### 1. 高性能

- **连接池**: 数据库连接池管理
- **查询优化**: 索引优化、批量操作
- **缓存策略**: 元数据缓存、查询结果缓存

### 2. 可扩展性

- **插件系统**: JavaScript插件支持
- **钩子系统**: 业务钩子扩展点
- **MCP协议**: Model Context Protocol支持

### 3. 可靠性

- **事务支持**: 完整的ACID事务
- **错误处理**: 完善的错误处理机制
- **日志系统**: 结构化日志和SQL日志

### 4. 安全性

- **JWT认证**: 基于JWT的用户认证
- **权限检查**: 细粒度的权限验证
- **数据隔离**: Schema级别的数据隔离

## 📊 数据模型层次

```
Space (空间)
  └── Base (基础)
        └── Table (表格)
              ├── Field (字段)
              ├── Record (记录)
              └── View (视图)
```
